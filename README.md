# Calyx

## Notes Before Release

### Cursor Click To Move (`Option+Click`)
- Calyx enables Ghostty's `cursor-click-to-move` feature.
- This feature depends on shell integration prompt markers and works best on primary-screen prompts.
- On Japanese/full-width text lines, cursor placement can be offset because movement is internally translated into arrow-key steps over terminal cells.
- If users work mainly with Japanese prompts, recommend disabling this option from `Calyx > Settings`.
